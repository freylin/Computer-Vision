1.Install VLFeat before running.
I have commended “run('vlfeat/toolbox/vl_setup’)”.

2.You can set “Mining_hard_negatives” to true to mine hard negatives.

3.You can use get_interesting_positive_features() function to get more positive features.

4.Extra datasets links:
Extra car train dataset (positive): http://www.robots.ox.ac.uk/~vgg/data/cars_brad/cars_brad.tar
Extra car train dataset (negative): http://www.robots.ox.ac.uk/~vgg/data/background/background.tar
Extra test dataset: http://www.robots.ox.ac.uk/~vgg/data/cars_markus/cars_markus.tar
Extra face train dataset(positive): http://www.robots.ox.ac.uk/~vgg/data/faces/faces.tar
